もじサンド プライバシーポリシー

第 1.2.0 版 ・ 制定日 2026-06-12 / 最終改定日 2026-06-14

「もじサンド」(以下「本アプリ」)は、ユーザーのプライバシーを尊重します。本ポリシーは、本アプリが取り扱う情報と、その利用方法を説明するものです。

サーバーに送信する情報

本アプリは、ゲームプレイに伴い以下の情報のみをサーバーに送信します。

  • 回答した単語
  • 出題されたお題(「あたま」と「おしり」の文字)
  • 回答した単語の文字数
  • 回答にかかった秒数
  • 各ラウンドの得点(デイリーチャレンジ)
  • ニックネーム(初回起動時に自動生成し、設定画面で変更できます)
  • 匿名のランダム識別子(端末ごとに生成する推測不能な符号。デイリーランキングで自分の順位を表示するためにのみ使います。氏名や端末そのものとは結び付きません)

氏名・メールアドレス・電話番号・ハードウェアの端末ID・位置情報など、あなたを直接特定できる情報は収集しません。アカウント登録もありません。

利用目的

送信された情報は、以下の目的にのみ利用します。

  • ゴースト対戦・デイリーランキング(他のプレイヤーの過去の回答との対戦・順位表示)の提供
  • 出題・判定の品質改善

第三者への送信・保存先

  • プレイ記録は Supabase(データセンター: 東京リージョン)に保存します。
  • 回答された単語が同梱辞書に未登録の場合のみ、その単語の妥当性判定のために OpenAI の API へ単語のみを送信します。判定結果はキャッシュされ、同じ単語が再度送信されることはありません。
  • 上記以外の第三者へ情報を提供することはありません。

IP アドレスの取り扱い

不正利用防止(連続投稿の制限)のため、リクエスト元 IP アドレスを salt 付きハッシュに変換した値のみを短期間保存します。

  • ハッシュ値から元の IP アドレスを復元することはできません。
  • 生の IP アドレスは保存しません。

端末内に保存する情報

以下の情報を端末内に保存します。ここで説明する送信情報を除き、これらが端末の外に送信されることはありません。

  • サウンドの ON/OFF 設定
  • ニックネーム
  • 遊び方ガイド(オンボーディング)を表示済みかどうか
  • 上記の匿名のランダム識別子
  • 自己ベストや対戦成績などのプレイ記録
  • デイリーチャレンジの結果と過去の履歴
  • ストアレビュー依頼を表示した回数

広告・トラッキング

本アプリは Google AdMob による広告を表示します。広告の配信・計測のため、広告 SDK が広告識別子(advertising identifier)などの情報を取得・利用することがあります。これらの情報は Google および広告配信に関わる事業者が、各社のプライバシーポリシーに従って取り扱います。

  • iOS では、App Tracking Transparency に従いトラッキングの許可を確認します。許可しない場合も広告は表示されますが、あなたに合わせた最適化(パーソナライズ)は行われません。
  • 欧州経済領域(EEA)・英国などでは、同意管理(Google User Messaging Platform)に従って広告に関する同意を取得します。
  • 本アプリは一般向け(児童を対象としない)として広告配信を設定しています。
  • Google が広告でデータをどう扱うかは、Google プライバシーポリシー(https://policies.google.com/privacy)および「広告における Google のデータ利用」(https://support.google.com/admob/answer/6128543)を参照してください。

広告 SDK 以外の解析 SDK は組み込んでいません。アプリ内で表示するストアレビューの依頼は OS の標準機能を利用するもので、当社サーバーへ情報を送信しません。

保存期間

プレイ記録はゴースト対戦・デイリーランキング機能の提供に必要な期間保存します。IP アドレスのハッシュ値はレート制限の判定に必要な短期間のみ保持し、その後削除されます。

本ポリシーの変更

本ポリシーを変更する場合は、版番号と改定日を更新したうえで本ページにて告知します。重要な変更がある場合は、アプリ内でもお知らせします。各版の変更点は末尾の改定履歴に記載します。
